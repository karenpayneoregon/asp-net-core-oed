## Create razor project

From solution level

```
dotnet new webapp --name Demo1
```

## Trust the development certificate

```
dotnet dev-certs https --trust
```

## Run 

```
dotnet watch run
```

Press <kbd>ctrl</kbd> + <kbd>c</kbd> to stop

## View

https://localhost:5001/
